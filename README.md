# Icomoon Offline App

Unofficail icomoon offline app, works with any browser.

## Usage

- Install [Node.js](https://nodejs.org/en/)
- Open up command line inside `icomoon-offline` directory
- Run `npm start`
- Head to [http://localhost:3000](http://localhost:3000)
